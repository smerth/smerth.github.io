1.1 Introduction

These are some notes concering how to get up and running with the LCM Drupal 8 Scaffold.

From the Last Call Media github repo:

LCM Drupal 8 Scaffold is a boilerplate Drupal 8 build that bundles some standard tools to make it a good starting point for an enterprise scale Drupal build. It is conceptually similar to drupal-composer/drupal-scaffold, but it has a much simpler (and more manual) Composer setup, and includes additional tools. For additional information on this project, see the 2016 Badcamp presentation slides
