Functions start with the keyword "on" followed by the "function name" followed by "parenthesis"

on addNums(a, b)
    return a + b
end addNums

of course after defining the function above, you must call it and pass in the expected arguments.

addNums(2, 3)

Passing in variables. I the example below variables are defined for paths and files and then these can be passed into functions.

tell application "Finder"
    set hasPS to get exists folder ((path to applications folder as string) & "Adobe Photoshop CS6")
    set theImagesFolder to ((path to desktop folder) & "Images" as string as alias)
    set theImages to the name of every file of theImagesFolder
end tell
 
on addComment(theImage, theImagesFolder)
    -- add a comment to a file
    -- note theImagesFolder must be passed in because it is defined outside the scope of the function
    set theFile to (theImagesFolder & theImage) as string as alias
    tell application "Finder"
        open theFile
    end tell
end addComment
 
on addWatermark(theImage)
    -- add a watermark to a file using photoshop
end addWatermark
 
if hasPS is false then
    -- no photoshop
    repeat with anImage in theImages
        addComment(anImage, theImagesFolder)
    end repeat
else
    -- has photshop
    repeat with anImage in theImages
        addComment(anImage, theImagesFolder)
        addWatermark(anImage)
    end repeat
end if

Note theImagesFolder must be passed into the function above because it is defined outside the scope of that function. To get around this you can use global variables.

6.0.1 Setting global variables

We can set a global variable of the path to the images folder because it doesn't change anywhere in the program.

global theImagesFolder
 
tell application "Finder"
 
    set hasPS to get exists folder ((path to applications folder as string) & "Adobe Photoshop CS6")
    set theImagesFolder to ((path to desktop folder) & "Images" as string as alias)
    set theImages to the name of every file of theImagesFolder
end tell
 
on addComment(theImage)
    -- add a comment to a file
    set theFile to (theImagesFolder & theImage) as string as alias
    tell application "Finder"
        open theFile
    end tell
end addComment
 
on addWatermark(theImage)
    -- add a watermark to a file using photoshop
end addWatermark
 
if hasPS is false then
    -- no photoshop
    repeat with anImage in theImages
        addComment(anImage)
    end repeat
else
    -- has photshop
    repeat with anImage in theImages
        addComment(anImage)
        addWatermark(anImage)
    end repeat
end if
