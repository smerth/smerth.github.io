About this site

This site is the result of a mash-up of Easybook and Foundation 6. Easybook is a Symfony application written in PHP. Foundation 6 is a useful markup and css framework for fast prototyping of responsive sites.

Easybook output can take the form of an epub book, a mobi book, a pdf, a webpage or chunked html. Because it is based on Symfony the output is passed through a template written in TWIG.

The input for Easybook is a set of markdown documents and associated image and resource files. Any of the above reading formats can be generated from a single cannonical set of markdown documents. These documents can be managed in git so that mulitple authors can work on the same documents with complete conifidence. Publishing mulitple versions of a book in different formats is a trivial affair and reconfiguring indices in also trivial. All this and more make Easybook a very flexible and powerful publishing platform.

Easybook allows you to write your own theme for any edition of a book. All you need to know is a bit about templating in general and TWIG in particular. You can write a theme that generates a complete website directly from Easybook output. Or, as in this case, it is possible to export chunked HTML markup from a custom theme in Easybook out into a separate Foundation site.

Zurb (the creators of Foundation) offer their own in-house Zurb-Template as an option to build a website. It comes with Zurb's templating library called Panini. Because Panini is based on Handlebars you have alot of flexibility to work with data using this template.

If you need help building a flexible, robust, publishing platform you can find me on github.
